# Gemini Discord Bot 🤖

A Discord bot integrated with Google's Gemini AI. Simply type `gemini <your prompt>` and the bot will respond with intelligent answers generated by Gemini!

## Features
- **Gemini 2.5 Flash**: Fast and intelligent responses using Google's generative AI model.
- **Long Message Handling**: Automatically splits responses exceeding Discord's 2000 character limit.
- **Easy Setup**: Secure configuration using `.env` files.

## Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nishad3107/Discord_Bot.git
   cd Discord_Bot
   ```

2. **Create a virtual environment & install dependencies**
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory and add your API keys:
   ```env
   DISCORD_TOKEN=your_discord_bot_token_here
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the bot**
   ```bash
   python main.py
   ```

## Cloud Hosting (24/7)
This bot is ready to be hosted on services like **Render**. 
1. Connect this repository to a Render **Background Worker**.
2. Add your `DISCORD_TOKEN` and `GEMINI_API_KEY` as Environment Variables in the Render dashboard.
3. Deploy!
